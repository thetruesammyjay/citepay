# Demo Script

## 1. Open the landing page

Show the premium fintech interface and explain the central idea:

> CitePay is an AI research assistant where every paid citation triggers a USDC micropayment to the publisher.

## 2. Run the query

Open `/query` and submit:

```text
Best stablecoin payment rails for African creators
```

Narrate the flow:

- Query intake
- Source discovery
- Trust scoring
- Payment decision
- USDC payment execution
- x402 source unlock
- Final answer generation

## 3. Show the dashboard

Open `/dashboard` and point out:

- Sources used
- Source trust score
- Payment amount
- Transaction hash
- Payment status
- Total spent
- Creator earnings
- Unlocked content count

## 4. Show publishers

Open `/publishers` and explain that publishers can set citation prices, connect wallets, and track revenue.

## 5. Close with the hackathon thesis

CitePay turns attribution into programmable revenue. AI stays useful, publishers get paid, and users get transparent answers with verifiable receipts.
